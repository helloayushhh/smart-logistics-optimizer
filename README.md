# 🚚 Logistics Optimizer

An intelligent Python-based logistics optimization system designed to improve **delivery route planning** and **order assignment** for truck fleets. By combining **machine learning**, **genetic optimization**, and **API-powered routing**, the system helps streamline logistics operations and improve delivery efficiency.

---

## ✨ Features

- 🚛 **Route Optimization**  
  Generates optimized delivery routes using the **GraphHopper API** to reduce travel time and improve operational efficiency.

- 📦 **Smart Order Assignment**  
  Assigns orders to trucks based on **mass** and **volume** constraints, ensuring optimal vehicle utilization.

- 🤖 **Machine Learning & AI**  
  Leverages **K-Means Clustering** and **Genetic Algorithms** to intelligently group deliveries and optimize dispatch planning.

- 🗄️ **Database Integration**  
  Connects with **MySQL** for efficient storage and real-time management of logistics data.

- 🗺️ **Interactive Route Visualization**  
  Displays optimized delivery routes on interactive maps using **Folium**, making route analysis simple and intuitive.

---

## 🛠️ Tech Stack

- **Language:** Python
- **Machine Learning:** K-Means Clustering
- **Optimization:** Genetic Algorithm
- **Routing API:** GraphHopper API
- **Database:** MySQL
- **Visualization:** Folium

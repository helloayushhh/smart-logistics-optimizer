# smart logistics optimizer

a python project that explores smarter delivery planning through route optimization, order assignment and machine learning.

---

## why i built this

i wanted to understand how logistics systems make delivery decisions.

this project combines routing, clustering and optimization to simulate a smarter delivery workflow while exploring practical machine learning concepts.

---

## features

- optimize delivery routes
- assign orders based on truck capacity
- cluster deliveries using k-means
- improve planning with genetic algorithms
- visualize routes on an interactive map
- store logistics data with mysql

---

## built with

- python
- mysql
- scikit-learn
- graphhopper api
- folium
